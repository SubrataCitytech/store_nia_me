## ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/SubrataCitytech/store_nia_me.git
```

2. Navigate to project folder:

```bash
cd store_nia_me
```

3. Install dependencies:

```bash
flutter pub get
```

4. Run the app:

```bash
flutter run
```
